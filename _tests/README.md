# Test Files Directory

This directory contains all test and diagnostic files for the Money Tracker system.

## Test Files

### Main Test Suites
- **MASTER_TEST_SUITE.js** - Comprehensive test suite for all system components
- **COMPLETE_SYSTEM_TEST.js** - Complete system integration tests
- **COMPREHENSIVE_TEST.js** - Comprehensive functionality tests
- **TEST_ALL_SYSTEMS.js** - Tests for all system modules

### Specialized Tests
- **TEST_WEB_APP.js** - Web application interface tests
- **TEST_SCRIPT.js** - Script functionality tests
- **DIAGNOSTIC_TEST.js** - System diagnostic tests
- **DIAGNOSTIC_TEST.js.bak** - Backup of diagnostic tests
- **AUTO_TEST_RUNNER.js** - Automated test runner

### Debug Tools
- **QUICK_DEBUG.js** - Quick debugging utilities
- **QUICK_DEBUG.js.bak** - Backup of debug utilities
- **AUTOMATED_VALIDATION.js** - Automated validation scripts

### Test Pages
- **auto_tests.html** - Automated testing interface
- **debug_api_test.html** - API debugging interface
- **test_report.html** - Test results reporting

### Documentation
- **AUTOMATED_TESTING.md** - Automated testing guide and documentation

## Running Tests

To run tests, you can execute them individually from the Apps Script editor or use the automated test runner.

**Note:** These files are kept separate from the main codebase to maintain clarity and organization.
